deploy 阶段总是使用的旧镜像 #

检查/helm/${project_name}/values-*.yaml文件，nodeSelector.appenv为test时，不要配置nodeSelector.nodetype。

nodeSelector:
  appenv: test

# or

nodeSelector:
  appenv: production
  nodetype: front

values.container.command 未执行 #

检查/helm/${project_name}/templates/deployment.yaml文件：

spec:
  template:
    spec:
      containers:
        {{- if .Values.container.command }}
        command: ["bash", "-c", {{ .Values.container.command }}]
        {{- end }}
